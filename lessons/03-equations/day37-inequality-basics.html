Key Idea (Inequality Basics):
Inequalities compare two expressions using <, ≤, >, or ≥. The solution is often a range of values, not just one number.

Example (General Algebra)

Explain and represent the inequality: x ≥ 5

Meaning: x is any number greater than or equal to 5.

Number line idea:

  • Closed dot at 5 (because 5 is included).
  • Arrow to the right to show all numbers greater than 5.

Check some values:

  • x = 5 → 5 ≥ 5 ✔
  • x = 7 → 7 ≥ 5 ✔
  • x = 3 → 3 ≥ 5 ✘

Your Turn (General Algebra)

Describe the solution set of x < −2 in words.

Show answer

The solution is all real numbers that are less than −2. On a number line, you would use an open circle at −2 and shade to the left.

STAAR-Style Practice

1. Which inequality represents “a number n is at most 12”?

  1. n < 12
  2. n ≤ 12
  3. n ≥ 12
  4. n > 12

2. A student must have more than 70 points to pass a quiz. Which inequality shows the possible scores s?

  1. s ≥ 70
  2. s > 70
  3. s ≤ 70
  4. s < 70

3. Which symbol makes the statement true? −4 ___ −1

  1. >
  2. ≥
  3. <
  4. ≤
Show STAAR-style answers and explanations

1. “At most 12” means 12 or less → ≤ → B.

2. “More than 70” means greater than 70, but not equal → s > 70 → B.

3. On a number line, −4 is to the left of −1, so −4 is less than −1 → use < → C.

Words like “at least,” “at most,” “no more than,” and “no less than” tell you whether to use ≤ or ≥ instead of just < or >.
