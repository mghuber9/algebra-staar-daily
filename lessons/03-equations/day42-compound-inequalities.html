Day 42 – Compound Inequalities

Goal: Practice skills with Compound Inequalities.

Key Idea (Compound Inequalities):
Compound inequalities use the words “and” or “or.” “And” means values that satisfy both inequalities (overlap). “Or” means values that satisfy at least one of the inequalities.

Example (General Algebra)

Solve and graph: −1 ≤ x + 2 < 5

Step 1 – Treat it like a 3-part inequality:

  • −1 ≤ x + 2 < 5

Step 2 – Subtract 2 from all three parts:

  • −1 − 2 ≤ x + 2 − 2 < 5 − 2
  • −3 ≤ x < 3

Meaning: x is between −3 and 3, including −3 but not including 3.

Your Turn (General Algebra)

Solve and describe: 1 < x + 4 ≤ 7

Show answer

Subtract 4 from each part: 1 − 4 < x + 4 − 4 ≤ 7 − 4.
−3 < x ≤ 3.
So x is greater than −3 and less than or equal to 3.

STAAR-Style Practice

1. Which inequality represents “x is greater than −2 and less than or equal to 5”?

  1. −2 ≥ x ≥ 5
  2. −2 < x ≤ 5
  3. −2 ≤ x < 5
  4. x < −2 or x ≤ 5

2. Solve: 2x − 1 ≥ 5 or x + 3 < 1

  1. x ≥ 3 or x < −2
  2. x ≥ 3 and x < −2
  3. x ≤ 3 or x > −2
  4. x ≤ 3 and x > −2

3. The solution to a compound inequality is −4 ≤ x ≤ 2. Which statement is true?

  1. x can only be −4 or 2.
  2. x can be any value less than −4 or greater than 2.
  3. x is between −4 and 2, including both −4 and 2.
  4. x is between −4 and 2, not including −4 or 2.
Show STAAR-style answers and explanations

1. “Greater than −2 and less than or equal to 5” → −2 < x ≤ 5 → B.

2. 2x − 1 ≥ 5 → 2x ≥ 6 → x ≥ 3. For x + 3 < 1 → x < −2. Combine with “or”: x ≥ 3 or x < −2 → A.

3. −4 ≤ x ≤ 2 means x is any value from −4 to 2, including endpoints → C.

“And” usually gives you a middle interval (between two numbers). “Or” often gives you two separate rays going in opposite directions.
