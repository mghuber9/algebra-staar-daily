Day 44 – Absolute Value Inequalities

Goal: Practice skills with Absolute Value Inequalities.

← Back to all lessons
Key Idea (Absolute Value Inequalities):
For |x − a| < b, think “distance from a is less than b” → an and inequality. For |x − a| > b, think “distance from a is greater than b” → an or inequality.

Example (General Algebra)

Solve: |x − 2| ≤ 3

Interpretation: The distance from 2 is at most 3.

Step 1 – Write as a compound inequality:

  • −3 ≤ x − 2 ≤ 3

Step 2 – Add 2 to all three parts:

  • −3 + 2 ≤ x − 2 + 2 ≤ 3 + 2
  • −1 ≤ x ≤ 5

Meaning: x is between −1 and 5, inclusive.

Your Turn (General Algebra)

Solve: |x + 1| < 4

Show answer

|x + 1| < 4 → −4 < x + 1 < 4.
Subtract 1: −5 < x < 3.
So x is between −5 and 3, not including endpoints.

STAAR-Style Practice

1. Solve: |x − 4| ≥ 2

  1. x ≤ 2 or x ≥ 6
  2. x ≥ 2 and x ≤ 6
  3. 2 ≤ x ≤ 6
  4. 2 < x < 6

2. The inequality |x − 5| < 1 describes numbers whose distance from 5 is less than 1. Which interval represents the solution?

  1. x < 4
  2. 4 < x < 6
  3. 4 ≤ x ≤ 6
  4. x > 6

3. Which inequality describes all numbers at least 3 units away from 0?

  1. |x| ≤ 3
  2. |x| < 3
  3. |x| ≥ 3
  4. |x| < −3
Show STAAR-style answers and explanations

1. |x − 4| ≥ 2 → x − 4 ≥ 2 or x − 4 ≤ −2 → x ≥ 6 or x ≤ 2 → A.

2. |x − 5| < 1 → −1 < x − 5 < 1 → add 5: 4 < x < 6 → B.

3. “At least 3 units away” means distance ≥ 3 → |x| ≥ 3 → C.

Less than (<) → think “between” (and). Greater than (>) → think “outside” (or).
