Day 35 – Literal Equations

Goal: Practice skills with Literal Equations.

← Back to all lessons
Key Idea (Literal Equations):
Literal equations have multiple variables. To solve for one variable, treat the others like constants and use inverse operations to isolate the target variable.

Example (General Algebra)

Solve the formula A = lw for w.

Step 1 – Identify what you are solving for: Solve for w.

Step 2 – Use inverse operations: A = lw means A = l·w. To isolate w, divide both sides by l.

  • A ÷ l = lw ÷ l
  • A / l = w

Final form: w = A / l

Your Turn (General Algebra)

Solve the equation P = 2l + 2w for w.

Show answer

P = 2l + 2w.
Subtract 2l: P − 2l = 2w.
Divide by 2: (P − 2l) / 2 = w.
Answer: w = (P − 2l) / 2.

STAAR-Style Practice

1. Solve the equation d = rt for t.

  1. t = dr
  2. t = d − r
  3. t = d / r
  4. t = r / d

2. The formula for the area of a triangle is A = (1/2)bh. Which expression shows the height h in terms of A and b?

  1. h = 2A / b
  2. h = A / (2b)
  3. h = b / (2A)
  4. h = (A − b) / 2

3. Solve the equation y = mx + b for x.

  1. x = (y − b) / m
  2. x = (y + b) / m
  3. x = my − b
  4. x = y / m − b
Show STAAR-style answers and explanations

1. d = rt → divide both sides by r: t = d / r → C.

2. A = (1/2)bh → multiply both sides by 2: 2A = bh → divide by b: h = 2A / b → A.

3. y = mx + b → subtract b: y − b = mx → divide by m: x = (y − b) / m → A.

Think of literal equations as “solving for a letter” instead of a number—the same solving rules still apply.
