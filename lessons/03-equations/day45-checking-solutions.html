Key Idea (Checking Solutions):
To check a solution, substitute it back into the original equation or inequality. If the statement is true, the solution works; if it’s false, it does not.

Example (General Algebra)

Check whether x = 4 is a solution to the equation 3x − 2 = 10.

Step 1 – Substitute x = 4:

  • 3(4) − 2 = 12 − 2 = 10

Step 2 – Compare: The left side equals the right side (10 = 10), so x = 4 is a solution.

Your Turn (General Algebra)

Is x = −1 a solution to 2x + 5 ≥ 3?

Show answer

Substitute x = −1: 2(−1) + 5 ≥ 3 → −2 + 5 ≥ 3 → 3 ≥ 3 ✔.
Yes, x = −1 is a solution.

STAAR-Style Practice

1. Which value of x is a solution to 4x + 1 = 17?

  1. x = 3
  2. x = 4
  3. x = 5
  4. x = 6

2. Does x = 2 satisfy the inequality 5x − 7 < 4?

  1. Yes, because 5(2) − 7 = 3 and 3 < 4.
  2. Yes, because 5(2) − 7 = 5 and 5 < 4.
  3. No, because 5(2) − 7 = 3 and 3 > 4.
  4. No, because 5(2) − 7 = 5 and 5 > 4.

3. A student claims that x = 1 is a solution to |x − 3| = 2. Is the student correct?

  1. Yes, because |1 − 3| = 2
  2. Yes, because |1 − 3| = −2
  3. No, because |1 − 3| = 4
  4. No, because |1 − 3| = 0
Show STAAR-style answers and explanations

1. 4x + 1 = 17 → 4x = 16 → x = 4. Only x = 4 works → B.

2. Plug in x = 2: 5(2) − 7 = 10 − 7 = 3. Check 3 < 4 → true → A.

3. |1 − 3| = |−2| = 2, so the equation is true → A.

Checking your solution takes just a few extra seconds and can catch sign mistakes or arithmetic errors before you move on.
