Day 43 – Absolute Value Equations

Goal: Practice skills with Absolute Value Equations.

← Back to all lessons
Key Idea (Absolute Value Equations):
For equations of the form |x − a| = b, think “distance from a is b.” This usually gives two solutions: x − a = b and x − a = −b, as long as b ≥ 0.

Example (General Algebra)

Solve: |x − 3| = 5

Step 1 – Split into two equations:

  • x − 3 = 5   or   x − 3 = −5

Step 2 – Solve each:

  • x − 3 = 5 → x = 8
  • x − 3 = −5 → x = −2

Check quickly:

  • |8 − 3| = |5| = 5 ✔
  • |−2 − 3| = |−5| = 5 ✔

Your Turn (General Algebra)

Solve: |x + 4| = 2

Show answer

x + 4 = 2 or x + 4 = −2.
From x + 4 = 2 → x = −2.
From x + 4 = −2 → x = −6.
Solutions: x = −2 and x = −6.

STAAR-Style Practice

1. What are the solutions to: |x| = 7?

  1. x = 7 only
  2. x = −7 only
  3. x = −7 and x = 7
  4. No solution

2. Solve: |2x − 1| = 3

  1. x = 1 only
  2. x = −1 only
  3. x = −1 and x = 2
  4. x = 1 and x = 2

3. Which equation has no solution?

  1. |x + 2| = 0
  2. |x − 5| = −1
  3. |x| = 3
  4. |x − 1| = 4
Show STAAR-style answers and explanations

1. |x| = 7 → x = 7 or x = −7 → C.

2. 2x − 1 = 3 or 2x − 1 = −3.
From 2x − 1 = 3 → 2x = 4 → x = 2.
From 2x − 1 = −3 → 2x = −2 → x = −1.
Solutions: x = −1 and x = 2 → C.

3. Absolute value is never negative, so |x − 5| = −1 has no solution → B.

If an absolute value equals a negative number, you can stop immediately—there is no real solution.
