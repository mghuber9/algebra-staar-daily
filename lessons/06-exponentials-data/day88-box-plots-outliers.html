Key Idea (Box Plots & Outliers):
A box plot uses the 5-number summary: • Minimum • Q1 (lower quartile) • Median • Q3 (upper quartile) • Maximum Outliers are values far outside the typical range.

Example (General Algebra)

Data: 2, 4, 5, 7, 8, 10, 13

  • Min = 2
  • Q1 = 4
  • Median = 7
  • Q3 = 10
  • Max = 13

Your Turn (General Algebra)

Data: 3, 3, 6, 7, 12

Show answer

Min = 3, Q1 = 3, Median = 6, Q3 = 7, Max = 12.

STAAR-Style Practice

1. In a box plot, what does the box represent?

  1. All data values
  2. Middle 50% of the data (Q1 to Q3)
  3. Maximum values only
  4. Outliers

2. Which value could be an outlier in the set: 10, 11, 12, 13, 14, 40?

  1. 10
  2. 11
  3. 14
  4. 40

3. The median represents:

  1. The most frequent value
  2. The average value
  3. The middle value
  4. The largest value
Show STAAR-style answers and explanations

1. Box = Q1 to Q3 → B

2. 40 is far from others → D

3. Middle value → C

Box plot = visual summary of spread; outliers stretch the “whiskers.”
