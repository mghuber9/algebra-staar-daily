Day 82 – Linear vs Exponential in Tables

Goal: Practice skills with Linear vs Exponential in Tables.

← Back to all lessons
Key Idea (Tables – Linear vs Exponential):
In a table with equal x-steps:
• Linear: y-values change by a constant difference (add/subtract same amount).
• Exponential: y-values change by a constant ratio (multiply by same amount).

Example (General Algebra)

Decide if each table is linear or exponential.

Table A

xy
04
17
210
313

Differences in y: +3, +3, +3 → constant difference → linear.

Table B

xy
05
110
220
340

Ratios in y: 10 ÷ 5 = 2, 20 ÷ 10 = 2, 40 ÷ 20 = 2 → constant ratio → exponential.
