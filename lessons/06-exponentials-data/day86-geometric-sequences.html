Key Idea (Geometric Sequences):
A geometric sequence multiplies by the same number each step. Formula: aₙ = a₁(r)ⁿ⁻¹
• a₁ = first term • r = common ratio

Example (General Algebra)

Find the 5th term of the sequence: 3, 6, 12, 24, …

Step 1 – Identify a₁ and r:

  • a₁ = 3
  • r = 2

Step 2 – Use formula:

  • a₅ = 3(2)⁴ = 3(16) = 48

Your Turn (General Algebra)

Find the 4th term of the sequence: 16, 8, 4, …

Show answer

a₁ = 16, r = 1/2.
a₄ = 16(1/2)³ = 16(1/8) = 2.

STAAR-Style Practice

1. Which is a geometric sequence?

  1. 1, 4, 7, 10
  2. 5, 10, 20, 40
  3. 2, 5, 8, 11
  4. 10, 8, 6, 4

2. What is the common ratio in 9, 3, 1, 1/3, … ?

  1. −3
  2. 3
  3. 1/3
  4. −1/3

3. Find the 6th term when a₁ = 2 and r = 3.

  1. 162
  2. 243
  3. 486
  4. 729
Show STAAR-style answers and explanations

1. Multiplies by 2 → B

2. 3 → 1 → 1/3 (÷3 each time) → ratio = 1/3 → C

3. a₆ = 2(3)⁵ = 2(243) = 486 → C

Geometric → multiply, NOT add.
