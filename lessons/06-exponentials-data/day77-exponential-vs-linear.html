Day 77 – Exponential vs Linear

Goal: Practice skills with Exponential vs Linear.

← Back to all lessons
Key Idea (Exponential vs Linear):
Linear functions change by adding/subtracting the same amount each step.
Exponential functions change by multiplying by the same amount (same ratio) each step.

Example (General Algebra)

Decide whether each situation is linear or exponential:

  • Situation A: You earn $50 per week for mowing lawns.
  • Situation B: A bacteria population doubles every hour.

Situation A:

  • Each week you add the same amount: +$50.
  • This is a constant difference → linear.

Situation B:

  • Each hour the population is multiplied by 2.
  • This is a constant ratio → exponential.

Your Turn (General Algebra)

A savings account grows by 3% each year. Is this linear or exponential?

Show answer

A percent growth repeats by multiplying by the same factor (like 1.03) each year → exponential.

STAAR-Style Practice

1. Which equation represents a linear function?

  1. y = 3x + 7
  2. y = 2x
  3. y = 5(1.1)x
  4. y = 0.5·3x

2. Which function is exponential?

  1. y = −4x + 2
  2. y = x² − 3
  3. y = 7(1.2)x
  4. y = 3x − 5

3. A table shows a sequence of values:

Is this relationship linear or exponential?

  1. Linear, because y increases by 6 each time.
  2. Linear, because x increases by 1.
  3. Exponential, because y is multiplied by 3 each time.
  4. Exponential, because y increases then decreases.
Show STAAR-style answers and explanations

1. Linear has the form y = mx + b → y = 3x + 7 → A.

2. Exponential has a variable in the exponent → 7(1.2)x → C.

3. y-values: 3, 9, 27, 81 → each time multiply by 3 → exponential → C.

Quick test: “same difference” → linear. “Same multiplier/ratio” → exponential.
