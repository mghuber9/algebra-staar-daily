Key Idea (Regression Meaning):
A regression line (line of best fit) shows the overall trend in bivariate data. • The slope tells how y changes as x increases. • The y-intercept tells the predicted starting value when x = 0.

Example (General Algebra)

A line of best fit for the data is y = 1.2x + 5.

  • Slope 1.2 → y increases by 1.2 units for every 1 increase in x.
  • Intercept 5 → predicted y-value when x = 0.

Your Turn (General Algebra)

The regression line is y = −0.4x + 10. What does the slope mean?

Show answer

y decreases by 0.4 for every 1-unit increase in x.

STAAR-Style Practice

1. If the regression line is y = 3x + 2, what does the 3 represent?

  1. y-intercept
  2. Starting value
  3. Rate of change
  4. Correlation value

2. What does the 2 represent in the same line?

  1. Starting value when x = 0
  2. Rate of change
  3. Median value
  4. Outlier value

3. If the slope of a regression line is negative, what does that indicate?

  1. As x increases, y increases.
  2. As x increases, y decreases.
  3. No relationship exists.
  4. There are no outliers.
Show STAAR-style answers and explanations

1. Slope = rate of change → C

2. Intercept = value when x = 0 → A

3. Negative slope → decreasing trend → B

A regression line is a prediction tool—not exact, but best-fit.
