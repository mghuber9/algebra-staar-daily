Goal: Practice skills with Decay Formula.

Key Idea (Exponential Decay):
Exponential decay can be modeled by y = a(1 − r)t, where:
• a = starting amount
• r = decay rate (decimal)
• t = number of time periods

Example (General Algebra)

A car’s value is $20,000 and decreases by 12% each year. Write a model and estimate the value after 2 years.

Step 1 – Identify a and r:

  • a = 20,000
  • r = 12% = 0.12

Step 2 – Write the model:

  • V(t) = 20,000(1 − 0.12)t = 20,000(0.88)t

Step 3 – Value after 2 years:

  • V(2) = 20,000(0.88)² ≈ 20,000(0.7744) ≈ 15,488

After 2 years, the car is worth about $15,488.

Your Turn (General Algebra)

A phone is worth $600 and loses 20% of its value each year. Write a function for the value V(t).

Show answer

a = 600, r = 0.20.
V(t) = 600(0.8)t.

STAAR-Style Practice

1. Which function shows an initial amount of 500 and a decay rate of 5% per year?

  1. y = 500(1.05)t
  2. y = 500(0.95)t
  3. y = 500(0.05)t
  4. y = 500(5)t

2. A substance has mass modeled by M(t) = 80(0.9)t. Which statement is true?

  1. The mass grows by 10% each year.
  2. The mass decays by 10% each year.
  3. The mass decays by 90% each year.
  4. The mass stays constant.

3. A video game loses 15% of its value each year. Which expression represents its value after 3 years if it starts at $40?

  1. 40(1.15)³
  2. 40(0.85)³
  3. 40(0.15)³
  4. 40 + 0.15(3)
Show STAAR-style answers and explanations

1. Decay factor = 1 − 0.05 = 0.95 → B.

2. Factor 0.9 = 1 − 0.1 → decays 10% per year → B.

3. a = 40, r = 0.15 → decay factor 0.85 → 40(0.85)³ → B.

For decay, the factor is less than 1: 1 − r. Every step you multiply by that factor instead of subtracting repeatedly.
