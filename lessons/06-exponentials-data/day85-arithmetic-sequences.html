Key Idea (Arithmetic Sequences):
An arithmetic sequence adds the same number each step. Formula: aₙ = a₁ + (n − 1)d
• a₁ = first term • d = common difference

Example (General Algebra)

Find the 10th term of the sequence: 4, 9, 14, 19, …

Step 1 – Identify a₁ and d:

  • a₁ = 4
  • d = 5

Step 2 – Use formula:

  • a₁₀ = 4 + (10 − 1)(5) = 4 + 45 = 49

Your Turn (General Algebra)

Find the 6th term of the sequence: 12, 7, 2, …

Show answer

a₁ = 12, d = −5.
a₆ = 12 + (6 − 1)(−5) = 12 − 25 = −13.

STAAR-Style Practice

1. Which sequence is arithmetic?

  1. 2, 4, 8, 16, …
  2. 5, 11, 17, 23, …
  3. 3, 9, 27, 81, …
  4. 1, 1, 2, 3, 5, …

2. What is the common difference in the sequence 10, 6, 2, −2, … ?

  1. −2
  2. 4
  3. −4
  4. 2

3. Find the 7th term of the arithmetic sequence with a₁ = 3 and d = 9.

  1. 48
  2. 57
  3. 60
  4. 63
Show STAAR-style answers and explanations

1. Only B adds the same amount (+6) → B

2. Differences: −4 each step → C

3. a₇ = 3 + 6(9) = 3 + 54 = 57 → B

Arithmetic → “add the same number.” Exponential → “multiply the same number.”
