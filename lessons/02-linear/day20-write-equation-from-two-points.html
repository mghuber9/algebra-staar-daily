Day 20 – Write Equation From Two Points

Goal: Practice skills with Write Equation From Two Points.

← Back to all lessons
Key Idea (Equation from Two Points):
To write y = mx + b from two points:
  1. Find the slope using m = (y₂ − y₁)/(x₂ − x₁).
  2. Use one point and the slope in y = mx + b to solve for b.
  3. Write the equation.

Example (General Algebra)

Write the equation of the line that passes through (1, 4) and (5, 12).

Step 1 – Find slope:

  • m = (12 − 4) / (5 − 1) = 8/4 = 2

Step 2 – Use y = mx + b with point (1, 4):

4 = 2(1) + b → 4 = 2 + b → b = 2.

Step 3 – Equation: y = 2x + 2.

Your Turn (General Algebra)

Write the equation of the line that passes through (−2, −1) and (2, 7).

Show answer

m = (7 − (−1)) / (2 − (−2)) = 8/4 = 2.
Use (−2, −1): −1 = 2(−2) + b → −1 = −4 + b → b = 3.
Equation: y = 2x + 3.

STAAR-Style Practice

1. Which equation represents the line that passes through (0, −3) and (4, 5)?

  1. y = 2x − 3
  2. y = x − 3
  3. y = 2x + 3
  4. y = (1/2)x − 3

2. A line passes through (−1, 6) and (3, −2). What is the equation of the line?

  1. y = −2x + 4
  2. y = −2x + 6
  3. y = 2x + 4
  4. y = 2x − 6

3. The graph of a line contains the points (2, 1) and (6, 9). Which equation best represents this line?

  1. y = 2x − 3
  2. y = 2x + 3
  3. y = 3x − 5
  4. y = 3x − 1
Show STAAR-style answers and explanations

1. m = (5 − (−3)) / (4 − 0) = 8/4 = 2, b = −3 → y = 2x − 3 → A.

2. m = (−2 − 6) / (3 − (−1)) = −8/4 = −2. Use (−1, 6): 6 = −2(−1) + b → 6 = 2 + b → b = 4 → y = −2x + 4 → A.

3. m = (9 − 1) / (6 − 2) = 8/4 = 2. Use (2, 1): 1 = 2(2) + b → 1 = 4 + b → b = −3 → y = 2x − 3 → A.

Many students forget to plug back in to find b—remind them the process is “slope first, then plug in to solve for b.”
