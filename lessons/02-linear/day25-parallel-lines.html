Key Idea (Parallel Lines):
Two lines are parallel if they have the same slope and different y-intercepts.

Example (General Algebra)

Line 1: y = 3x + 1
Line 2: y = 3x − 4

Both lines have slope 3 → they are parallel. Different intercepts (1 and −4) mean they never cross.

Your Turn (General Algebra)

Is the line with equation y = −2x + 7 parallel to the line y = −2x − 5? Explain.

Show answer

Yes. Both have the same slope, −2, and different y-intercepts, so they are parallel.

STAAR-Style Practice

1. Which line is parallel to y = 4x − 3?

  1. y = −4x + 1
  2. y = 4x + 7
  3. y = (1/4)x − 3
  4. y = −(1/4)x − 3

2. A line has equation y = −3x + 2. Which equation represents a line parallel to it?

  1. y = −3x − 5
  2. y = 3x + 2
  3. y = −(1/3)x + 2
  4. y = 2x − 3

3. Which statement about the graphs of y = 2x + 3 and y = 2x − 4 is true?

  1. They intersect at exactly one point.
  2. They are the same line.
  3. They are parallel and never intersect.
  4. They are perpendicular.
Show STAAR-style answers and explanations

1. Parallel means same slope: 4. Only B has slope 4.

2. Parallel to y = −3x + 2 must have slope −3 → A is correct.

3. Both have slope 2, different intercepts, so they are parallel → C.

On STAAR, many questions say “line parallel to…” → immediately match the slope.
