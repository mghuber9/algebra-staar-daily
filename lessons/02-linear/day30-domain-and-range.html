Day 30 – Domain and Range

Goal: Practice skills with Domain and Range.

← Back to all lessons
Key Idea (Domain & Range):
  • Domain: all possible input values (x-values).
  • Range: all possible output values (y-values).
On STAAR, check whether the situation makes sense for negatives, fractions, etc.

Example (General Algebra)

A function represents the number of students in a class each year: 9th, 10th, 11th, and 12th grade.

Grade level (x) Number of students (y)
9 28
10 30
11 26
12 24

Domain: {9, 10, 11, 12}
Range: {28, 30, 26, 24}

Your Turn (General Algebra)

A function pairs each day of the week with the number of hours a store is open that day. Describe the domain and range in words.

Show answer

Domain: the days of the week (Monday through Sunday).
Range: the number of hours the store is open each of those days.

STAAR-Style Practice

1. The function C = 5n represents the cost C (in dollars) of buying n notebooks. Which set could represent the domain for this situation?

  1. {−2, −1, 0}
  2. {0.5, 1.5, 2.5}
  3. {0, 1, 2, 3, 4}
  4. {5, 10, 15, 20}

2. A function represents the height of a plant in centimeters after t weeks, written as h(t) = 3t + 5. If t is the number of weeks, which is the most reasonable domain?

  1. All real numbers
  2. All integers
  3. All nonnegative whole numbers
  4. All negative numbers

3. A function shows the relationship between the number of games a team wins and the total number of points they score. Which statement best describes the range?

  1. The set of possible numbers of games the team could play
  2. The set of possible numbers of points the team could score
  3. The set of possible days the team could practice
  4. The set of possible players on the team
Show STAAR-style answers and explanations

1. You can’t buy negative or fractional notebooks; 0,1,2,3,4 works as a reasonable domain → C.

2. Time in weeks can be 0,1,2,...; whole number weeks make sense → C (nonnegative whole numbers).

3. Range is the set of output values (total points scored) → B.

Always connect domain to “what can x be in this story?” and range to “what outputs are possible?”
