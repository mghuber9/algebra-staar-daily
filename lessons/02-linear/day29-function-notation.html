Day 29 – Function Notation

Goal: Practice skills with Function Notation.

← Back to all lessons
Key Idea (Function Notation):
If y = mx + b, we can also write f(x) = mx + b. f(x) is just a name for the output when the input is x.

Example (General Algebra)

A function is defined by f(x) = 2x − 3.

Find f(4).

Step 1 – Substitute: replace x with 4.

f(4) = 2(4) − 3 = 8 − 3 = 5.

Your Turn (General Algebra)

If g(x) = −3x + 7, find g(2).

Show answer

g(2) = −3(2) + 7 = −6 + 7 = 1.

STAAR-Style Practice

1. A function is defined by f(x) = 5x + 1. What is f(3)?

  1. 6
  2. 16
  3. 18
  4. 5

2. A function h is defined by h(x) = 4 − x. Which value is h(−2)?

  1. 6
  2. −6
  3. 2
  4. −2

3. The function p(x) = 3x − 5 gives the profit p(x) in dollars when x items are sold. What does p(10) represent?

  1. The profit when 3 items are sold
  2. The profit when 5 items are sold
  3. The profit when 10 items are sold
  4. The number of items sold when the profit is 10
Show STAAR-style answers and explanations

1. f(3) = 5(3) + 1 = 15 + 1 = 16 → B.

2. h(−2) = 4 − (−2) = 4 + 2 = 6 → A.

3. p(10) is the profit when x = 10 → profit when 10 items are sold → C.

Remind students: f(x) is like “y,” but it also reminds you what the input is.
