Day 18 – Write Equation From Table

Goal: Practice skills with Write Equation From Table.

← Back to all lessons
Key Idea (Equation from a Table):
From a table for a linear function:
  1. Find the slope (rate of change).
  2. Use a point and y = mx + b to solve for b.
  3. Write the equation y = mx + b.

Example (General Algebra)

The table shows a linear relationship between x and y.

x y
0 3
2 9

Write an equation that represents this relationship.

Step 1 – Find slope: Use (0, 3) and (2, 9).

  • Δy = 9 − 3 = 6
  • Δx = 2 − 0 = 2
  • m = 6 ÷ 2 = 3

Step 2 – Use y = mx + b with (0, 3):

3 = 3(0) + b → b = 3.

Step 3 – Write equation: y = 3x + 3.

Your Turn (General Algebra)

Use the table to write an equation.

x y
1 4
3 10
Show answer

Δy = 10 − 4 = 6, Δx = 3 − 1 = 2 → m = 3.
Use (1, 4): 4 = 3(1) + b → b = 1.
Equation: y = 3x + 1.

STAAR-Style Practice

1. The table shows the cost y (in dollars) of x notebooks.

x y
2 6
5 15

Which equation represents this situation?

  1. y = 2x + 2
  2. y = 3x
  3. y = 3x + 2
  4. y = 2x + 3

2. A relationship is shown in the table.

x y
0 −4
4 4

What is the equation of this line?

  1. y = 2x − 4
  2. y = 2x + 4
  3. y = x − 4
  4. y = x + 4

3. The table shows the total charge y for x hours of tutoring.

x (hours) y (dollars)
1 35
3 95

Which equation best models this situation?

  1. y = 30x + 5
  2. y = 35x + 5
  3. y = 30x + 35
  4. y = 35x + 25
Show STAAR-style answers and explanations

1. Δy = 15 − 6 = 9, Δx = 5 − 2 = 3 → m = 3. Check (2, 6): 6 = 3(2) = 6 → B (no intercept term).

2. (0, −4) and (4, 4): m = (4 − (−4)) / (4 − 0) = 8/4 = 2 → b = −4 → y = 2x − 4 → A.

3. (1, 35) and (3, 95): Δy = 60, Δx = 2 → m = 30. Use (1, 35): 35 = 30(1) + b → b = 5 → y = 30x + 5 → A.

Many STAAR questions expect students to interpret both the slope (rate) and intercept in the context of a table.
