Day 28 – Linear vs Nonlinear

Goal: Practice skills with Linear vs Nonlinear.

← Back to all lessons
Key Idea (Linear vs Nonlinear):
  • Linear: graph is a straight line; equation can be written in the form y = mx + b.
  • Nonlinear: graph is curved or has exponents, products of variables, etc.

Example (General Algebra)

Decide if each function is linear or nonlinear:

  • y = 3x − 5
  • y = x² + 1

y = 3x − 5 → linear (x to the first power, straight line).
y = x² + 1 → nonlinear (x² makes a parabola).

Your Turn (General Algebra)

Label each function as linear or nonlinear:

  • y = −4x
  • y = 2x³
Show answer

y = −4x → linear.
y = 2x³ → nonlinear (x³ gives a curved graph).

STAAR-Style Practice

1. Which of these equations represents a linear function?

  1. y = 2x² − 1
  2. y = 5x − 3
  3. y = x³ + 4
  4. y = 3/x

2. The table shows a function.

x y
0 2
1 5
2 8
3 11

Is this function linear or nonlinear?

  1. Linear, because y increases by the same amount each time.
  2. Nonlinear, because y increases.
  3. Linear, because y stays the same.
  4. Nonlinear, because x increases.

3. Which graph represents a nonlinear function?

  1. A straight line slanting up
  2. A straight horizontal line
  3. A curved line shaped like a U
  4. A vertical line
Show STAAR-style answers and explanations

1. y = 5x − 3 is linear (x to 1st power) → B.

2. y increases by 3 each time (2→5→8→11) → constant rate → linear → A.

3. A U-shaped curve is a parabola (x²), so nonlinear → C.

Shortcut: any x², x³, xy, 1/x, etc. → nonlinear. “Just x” to the first power → usually linear.
