Day 17 – Graphing With Slope-Intercept

Goal: Practice skills with Graphing With Slope-Intercept.

← Back to all lessons
Key Idea (Graphing y = mx + b):
In slope-intercept form, y = mx + b:
  • b = y-intercept (where the line crosses the y-axis).
  • m = slope (rise/run) to move from one point to another.

Example (General Algebra)

Graph the line y = 2x − 1.

  1. Plot the y-intercept (0, −1).
  2. Use the slope m = 2 = 2/1 → rise 2, run 1. From (0, −1), go up 2 and right 1 to (1, 1).
  3. Plot (1, 1) and draw a line through the points.

Your Turn (General Algebra)

Describe how you would graph y = −(1/2)x + 3.

Show answer

y-intercept: (0, 3).
Slope m = −1/2 → down 1, right 2 from each point. Plot (0, 3), then (2, 2), then (4, 1), and draw the line.

STAAR-Style Practice

1. Which statement is true about the graph of y = 3x + 4?

  1. It crosses the y-axis at (0, 3).
  2. It crosses the y-axis at (0, 4).
  3. It crosses the x-axis at (3, 0).
  4. It has slope −3.

2. A line has slope −2 and y-intercept 5. Which equation represents this line?

  1. y = −2x + 5
  2. y = 2x + 5
  3. y = −2x − 5
  4. y = 5x − 2

3. The graph of a line has points (0, −1) and (1, 2). Which equation matches this line?

  1. y = 2x − 1
  2. y = 3x − 1
  3. y = −2x − 1
  4. y = 2x + 1
Show STAAR-style answers and explanations

1. In y = 3x + 4, b = 4 → y-intercept is (0, 4) → B.

2. Slope = −2, y-intercept = 5 → y = −2x + 5 → A.

3. From (0, −1), b = −1. From (0, −1) to (1, 2): rise = 3, run = 1 → m = 3. So y = 3x − 1 → B.

Have students always identify m and b before touching the graph: “m is ___, b is ___”.
