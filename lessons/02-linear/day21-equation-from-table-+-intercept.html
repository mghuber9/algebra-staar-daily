Day 21 – Equation From Table + Intercept

Goal: Practice skills with Equation From Table + Intercept.

← Back to all lessons
Key Idea (Equation from Table + Intercept):
Use the table to find the slope, then look for the y-intercept (x = 0) if it is shown. If x = 0 is not given, use y = mx + b with any point to solve for b.

Example (General Algebra)

The table shows a linear function.

x y
0 −2
3 7

Write an equation in the form y = mx + b.

Step 1 – Find slope: Use (0, −2) and (3, 7).

  • Δy = 7 − (−2) = 9
  • Δx = 3 − 0 = 3
  • m = 9 ÷ 3 = 3

Step 2 – Identify y-intercept: At x = 0, y = −2 → b = −2.

Step 3 – Equation: y = 3x − 2.

Your Turn (General Algebra)

The table shows a linear function.

x y
0 4
5 14

Write the equation in y = mx + b form.

Show answer

m = (14 − 4) ÷ (5 − 0) = 10 ÷ 5 = 2.
b = 4 (from x = 0).
Equation: y = 2x + 4.

STAAR-Style Practice

1. The table represents a linear relationship.

x y
0 5
2 11

Which equation represents this relationship?

  1. y = 3x + 5
  2. y = 5x + 3
  3. y = 6x + 5
  4. y = 2x + 5

2. The table shows the cost y of x movie tickets.

x (tickets) y (dollars)
1 11
3 27

Which equation best models this situation?

  1. y = 8x + 3
  2. y = 9x + 2
  3. y = 5x + 6
  4. y = 4x + 7

3. A table shows a linear function where the y-intercept is −6 and the slope is 4. Which equation matches this function?

  1. y = 4x − 6
  2. y = −4x − 6
  3. y = 6x − 4
  4. y = 4x + 6
Show STAAR-style answers and explanations

1. m = (11 − 5) ÷ (2 − 0) = 6 ÷ 2 = 3, b = 5 → y = 3x + 5 → A.

2. m = (27 − 11) ÷ (3 − 1) = 16 ÷ 2 = 8. Use (1, 11): 11 = 8(1) + b → b = 3 → y = 8x + 3 → A.

3. Slope 4, y-intercept −6 → y = 4x − 6 → A.

Trick: if you see x = 0 in the table, you already know the y-intercept without extra work.
