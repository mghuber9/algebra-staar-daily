Day 69 – Solving by Factoring

Goal: Practice skills with Solving by Factoring.

Key Idea (Solving by Factoring):
To solve a quadratic equation, first set it equal to zero, factor, then use the Zero Product Property: if (expression)(expression) = 0, then each factor = 0.

Example (General Algebra)

Solve: x² − 5x + 6 = 0

Step 1 – Factor the quadratic:

  • x² − 5x + 6 = (x − 2)(x − 3)

Step 2 – Use Zero Product Property:

  • (x − 2)(x − 3) = 0 → x − 2 = 0 or x − 3 = 0
  • x = 2 or x = 3

Solutions: x = 2 and x = 3.

Your Turn (General Algebra)

Solve: x² + x − 12 = 0

Show answer

Factor: x² + x − 12 = (x + 4)(x − 3).
x + 4 = 0 → x = −4; x − 3 = 0 → x = 3.
Solutions: x = −4, x = 3.

STAAR-Style Practice

1. Solve: x² − 9 = 0

  1. x = 3 only
  2. x = −3 only
  3. x = −3 or x = 3
  4. No solution

2. Solve: 2x² + 3x − 2 = 0

  1. x = −2 or x = 1/2
  2. x = −1/2 or x = 2
  3. x = −2 or x = −1/2
  4. x = 1/2 or x = 2

3. Which is the first step when solving 5x² − 20 = 0 by factoring?

  1. Add 20 to both sides.
  2. Factor out the GCF.
  3. Multiply both sides by x.
  4. Divide both sides by 20.
Show STAAR-style answers and explanations

1. x² − 9 = (x − 3)(x + 3) → x = 3 or x = −3 → C.

2. Factor: 2x² + 3x − 2 = (2x − 1)(x + 2).
2x − 1 = 0 → x = 1/2; x + 2 = 0 → x = −2 → A.

3. 5x² − 20 = 0 → GCF is 5 → 5(x² − 4) = 0 → factoring GCF is first → B.

“Solve by factoring” always means: get 0 on one side → factor → set each factor equal to 0 → solve.
