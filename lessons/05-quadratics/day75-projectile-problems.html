Key Idea (Projectile Problems):
Height of a thrown object is often modeled by a quadratic like h(t) = −16t² + vt + s (feet), where t is time, v is initial velocity, and s is starting height.

Example (General Algebra)

A ball is thrown upward from a height of 5 feet with an initial velocity of 24 ft/s. Its height after t seconds is given by
h(t) = −16t² + 24t + 5.

Question: What is the maximum height the ball reaches?

Step 1 – Find the time at the vertex:

  • a = −16, b = 24 → t = −b / (2a) = −24 / (2·−16) = −24 / −32 = 0.75 seconds

Step 2 – Plug t = 0.75 into h(t):

  • h(0.75) = −16(0.75)² + 24(0.75) + 5
  • 0.75² = 0.5625
  • h ≈ −16(0.5625) + 18 + 5 = −9 + 23 = 14 feet

The maximum height is about 14 feet.

Your Turn (General Algebra)

A toy rocket is launched from the ground with height model h(t) = −16t² + 32t. At what time does it reach its maximum height?

Show answer

a = −16, b = 32 → t = −b / (2a) = −32 / (−32) = 1 second.
The rocket reaches maximum height at t = 1 second.

STAAR-Style Practice

1. A ball is tossed so that its height is modeled by h(t) = −16t² + 16t + 4. What does the number 4 represent?

  1. The maximum height
  2. The time when it lands
  3. The starting height
  4. The initial velocity

2. For the function in question 1, what is the time at which the ball reaches its maximum height?

  1. 0.25 seconds
  2. 0.5 seconds
  3. 1 second
  4. 2 seconds

3. What is the maximum height of the ball in question 1?

  1. 8 feet
  2. 12 feet
  3. 16 feet
  4. 20 feet
Show STAAR-style answers and explanations

For h(t) = −16t² + 16t + 4:

1. The constant term 4 is the height when t = 0 → starting height → C.

2. a = −16, b = 16 → t = −16 / (2·−16) = −16 / −32 = 0.5 → B.

3. h(0.5) = −16(0.5)² + 16(0.5) + 4 = −16(0.25) + 8 + 4 = −4 + 12 = 8 → A.

In projectile problems, the vertex gives the peak height; the constant term is the starting height; the t-intercepts tell when it hits the ground.
