Day 61 – Graphing Parabolas

Goal: Practice skills with Graphing Parabolas.

Key Idea (Graphing Parabolas):
To graph a quadratic, find key features: vertex, axis of symmetry, y-intercept, and a few symmetric points. Then sketch the U-shape (opening up or down).

Example (General Algebra)

Graph the quadratic: y = x² − 4x + 3

Step 1 – Find the vertex using x = −b / (2a):

  • a = 1, b = −4 → x = −(−4) / (2·1) = 4 / 2 = 2

Step 2 – Find the y-value of the vertex:

  • y = (2)² − 4(2) + 3 = 4 − 8 + 3 = −1
  • Vertex: (2, −1)

Step 3 – Find the y-intercept (when x = 0):

  • y = 0² − 4(0) + 3 = 3 → point (0, 3)

Step 4 – Use symmetry: The axis of symmetry is x = 2, so a matching point to (0, 3) is (4, 3).

Plot (2, −1), (0, 3), (4, 3), and connect with a smooth U-shaped curve opening upward.

Your Turn (General Algebra)

For y = x² + 2x + 1, identify the vertex and y-intercept before graphing.

Show answer

Vertex: x = −b / (2a) = −2 / (2·1) = −1.
y at x = −1: (−1)² + 2(−1) + 1 = 1 − 2 + 1 = 0 → vertex (−1, 0).
y-intercept: x = 0 → y = 0² + 2·0 + 1 = 1 → (0, 1).

STAAR-Style Practice

1. The graph of y = x² − 6x + 8 crosses the y-axis at which point?

  1. (0, −6)
  2. (0, 8)
  3. (6, 0)
  4. (8, 0)

2. Which statement is true about the graph of y = −x² + 4x − 1?

  1. The parabola opens upward.
  2. The parabola opens downward.
  3. The graph is a straight line.
  4. The vertex is at (0, −1).

3. The graph of a quadratic function has a vertex at (1, −4) and passes through (0, −1) and (2, −1). Which describes the graph?

  1. Opens upward and is symmetric about x = 0
  2. Opens upward and is symmetric about x = 1
  3. Opens downward and is symmetric about x = 0
  4. Opens downward and is symmetric about x = 1
Show STAAR-style answers and explanations

1. y-intercept is c → (0, 8) → B.

2. Leading coefficient is −1 (negative) → opens downward → B.

3. Points (0, −1) and (2, −1) are symmetric around x = 1, and vertex is lowest point → opens upward, symmetric about x = 1 → B.

Graphing a parabola is easier if you start with the vertex and axis of symmetry, then use symmetry to find matching points.
