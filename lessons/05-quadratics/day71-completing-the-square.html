Key Idea (Completing the Square):
For x² + bx, you can “complete the square” by adding (b/2)². This turns x² + bx into a perfect square trinomial (x + b/2)².

Example (General Algebra)

Rewrite x² + 6x in completed-square form.

Step 1 – Identify b: In x² + 6x, b = 6.

Step 2 – Compute (b/2)²:

  • b/2 = 6/2 = 3
  • (b/2)² = 3² = 9

Step 3 – Add 9 to complete the square:

  • x² + 6x + 9 = (x + 3)²

So x² + 6x can be written as (x + 3)² − 9 if you want an equivalent expression.

Your Turn (General Algebra)

What number should be added to x² − 8x to complete the square?

Show answer

b = −8 → b/2 = −4 → (b/2)² = (−4)² = 16.
You should add 16: x² − 8x + 16 = (x − 4)².

STAAR-Style Practice

1. To complete the square for x² + 10x, which value should be added?

  1. 5
  2. 10
  3. 25
  4. 100

2. When completing the square for x² + 4x, the trinomial becomes x² + 4x + 4. What is the factored form?

  1. (x + 2)(x + 4)
  2. (x + 2)²
  3. (x + 4)²
  4. (x − 2)²

3. Which expression is a completed square?

  1. x² + 7x + 10
  2. x² − 6x + 9
  3. x² − 3x + 7
  4. x² + 8x + 10
Show STAAR-style answers and explanations

1. b = 10 → (b/2)² = 5² = 25 → C.

2. x² + 4x + 4 = (x + 2)² → B.

3. x² − 6x + 9 = (x − 3)², a perfect square trinomial → B.

The “magic number” always comes from (b/2)² — memorize that pattern and completing the square becomes mechanical.
