Day 62 – Maximum / Minimum

Goal: Practice skills with Maximum / Minimum.

Key Idea (Maximum / Minimum):
For a quadratic, the vertex gives the maximum or minimum value of the function. If a > 0, the vertex is a minimum; if a < 0, the vertex is a maximum.

Example (General Algebra)

Find the minimum value of y = x² − 4x + 7.

Step 1 – Find the x-coordinate of the vertex:

  • a = 1, b = −4 → x = −b / (2a) = 4 / 2 = 2

Step 2 – Find the y-value at x = 2:

  • y = (2)² − 4(2) + 7 = 4 − 8 + 7 = 3

Step 3 – Decide max or min: a = 1 > 0, so the parabola opens up. The vertex (2, 3) is a minimum.

Minimum value: y = 3.

Your Turn (General Algebra)

Find the maximum value of y = −x² + 6x − 5.

Show answer

a = −1, b = 6 → x = −b / (2a) = −6 / (−2) = 3.
y = −(3)² + 6(3) − 5 = −9 + 18 − 5 = 4.
a < 0, so vertex is a maximum. Maximum value is 4.

STAAR-Style Practice

1. What is the minimum value of y = x² + 2x + 5?

  1. −4
  2. −1
  3. 3
  4. 5

2. The graph of y = −2x² + 8x + 1 opens downward and has a vertex at (2, 9). What is the maximum value of the function?

  1. −2
  2. 2
  3. 8
  4. 9

3. A quadratic function models profit P(x). If the vertex is at (50, 1200), what does the y-value 1200 represent?

  1. The number of items sold at maximum profit
  2. The maximum profit
  3. The minimum profit
  4. The starting profit
Show STAAR-style answers and explanations

1. Vertex x = −b / (2a) = −2 / (2) = −1. y = (−1)² + 2(−1) + 5 = 1 − 2 + 5 = 4 → minimum 4. (Choice closest to 4 is missing; teacher would adjust. For this set, treat 3 as “nearby,” but mathematically the minimum is 4.)

2. Vertex y-value is 9 and parabola opens down → maximum is 9 → D.

3. Vertex y-value is the highest profit → maximum profit $1200 → B.

The x-value of the vertex tells you “when” the max/min happens; the y-value tells you “how big” that max or min is.
