Day 57 – Standard Form

Goal: Practice skills with Standard Form.

← Back to all lessons
Key Idea (Standard Form):
The standard form of a quadratic is y = ax² + bx + c. The coefficient a tells you if the parabola opens up or down; c is the y-intercept.

Example (General Algebra)

Consider the quadratic y = 2x² − 4x + 3.

Identify a, b, and c:

  • a = 2
  • b = −4
  • c = 3

Interpret:

  • a = 2 > 0 → parabola opens upward.
  • c = 3 → the graph crosses the y-axis at (0, 3).

Your Turn (General Algebra)

For y = −x² + 6x − 1, identify a, b, and c.

Show answer

a = −1, b = 6, c = −1.

STAAR-Style Practice

1. Which equation is in standard form?

  1. y = (x − 3)(x + 2)
  2. y = 2(x + 1)² − 4
  3. y = −3x² + 5x − 7
  4. x² + 4x − 1 = 0

2. For the quadratic y = −2x² + 9x + 1, what is the y-intercept?

  1. (−2, 0)
  2. (0, −2)
  3. (0, 1)
  4. (1, 0)

3. In the quadratic y = ax² + bx + c, what does it mean if a is negative?

  1. The parabola opens upward.
  2. The parabola opens downward.
  3. The graph is a straight line.
  4. The graph is a circle.
Show STAAR-style answers and explanations

1. Standard form is y = ax² + bx + c → only y = −3x² + 5x − 7 matches → C.

2. y-intercept is c, when x = 0 → (0, 1) → C.

3. a < 0 means the parabola opens down (like a frown) → B.

Standard form is best for spotting the y-intercept quickly and using formulas like the axis of symmetry x = −b / (2a).
