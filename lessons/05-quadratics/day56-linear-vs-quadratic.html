Day 56 – Linear vs Quadratic

Goal: Practice skills with Linear vs Quadratic.

← Back to all lessons
Key Idea (Linear vs Quadratic):
Linear functions have the form y = mx + b and make straight lines. Quadratic functions have the form y = ax² + bx + c and make parabolas (U-shaped curves).

Example (General Algebra)

Decide whether each equation is linear or quadratic:

  • y = 3x − 4
  • y = 2x² − x + 1

First equation: y = 3x − 4

  • x is to the first power only → this is linear.

Second equation: y = 2x² − x + 1

  • It has an x² term → this is quadratic.

Your Turn (General Algebra)

Is y = −4x² + 9 linear or quadratic?

Show answer

There is an x² term and no higher powers → quadratic.

STAAR-Style Practice

1. Which function is linear?

  1. y = x² − 1
  2. y = −3x + 5
  3. y = 2x² + 4x
  4. y = −x² + 7

2. Which function is quadratic?

  1. y = 7 − 2x
  2. y = 3x
  3. y = x² + 4
  4. y = −1

3. A table of values for a function shows that the y-values increase by the same amount each time x increases by 1. Is the function most likely linear or quadratic?

  1. Linear
  2. Quadratic
  3. Neither
  4. Both
Show STAAR-style answers and explanations

1. Only y = −3x + 5 has x to the first power only → linear → B.

2. y = x² + 4 has x² → quadratic → C.

3. Constant first differences suggest a linear function → A.

Quick check: if you see x² (and not x³ etc.), think “quadratic.” If you only see x¹, think “linear.”
