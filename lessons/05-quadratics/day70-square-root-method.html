Day 70 – Square Root Method

Goal: Practice skills with Square Root Method.

← Back to all lessons
Key Idea (Square Root Method):
If an equation has the form x² = k (or (x − h)² = k), solve by taking square roots of both sides: x = ±√k. Don’t forget the ± when k > 0.

Example (General Algebra)

Solve: x² = 16

Step 1 – Take square roots of both sides:

  • x = ±√16
  • x = ±4

Solutions: x = 4 and x = −4.

Your Turn (General Algebra)

Solve: (x − 3)² = 9

Show answer

Take square roots: x − 3 = ±√9 = ±3.
Case 1: x − 3 = 3 → x = 6.
Case 2: x − 3 = −3 → x = 0.
Solutions: x = 6 and x = 0.

STAAR-Style Practice

1. Solve: x² = 25

  1. x = 5
  2. x = −5
  3. x = 0 or x = 5
  4. x = −5 or x = 5

2. Solve: 4(x + 1)² = 36

  1. x = −4 or x = 2
  2. x = −5 or x = 3
  3. x = −1 or x = 1
  4. x = −2 or x = 4

3. Solve: (x − 2)² = 0

  1. x = 0
  2. x = 2
  3. x = −2
  4. x = 0 or x = 2
Show STAAR-style answers and explanations

1. x² = 25 → x = ±√25 = ±5 → D.

2. 4(x + 1)² = 36 → (x + 1)² = 9 → x + 1 = ±3.
x + 1 = 3 → x = 2; x + 1 = −3 → x = −4 → A.

3. (x − 2)² = 0 → x − 2 = 0 → x = 2 (only one solution) → B.

The square root method works best when the x-term is already isolated in a perfect square, like x² or (x − h)².
