Key Idea (Discriminant):
The discriminant is D = b² − 4ac. It tells how many real solutions a quadratic has:
  • D > 0 → two real solutions
  • D = 0 → one real solution
  • D < 0 → no real solutions

Example (General Algebra)

For x² − 6x + 9 = 0, find the discriminant and number of solutions.

Step 1 – Identify a, b, c: a = 1, b = −6, c = 9.

Step 2 – Compute D = b² − 4ac:

  • D = (−6)² − 4·1·9 = 36 − 36 = 0

Step 3 – Interpret: D = 0 → the equation has one real solution (a repeated root).

Your Turn (General Algebra)

For 2x² + 3x + 5 = 0, compute the discriminant and state how many real solutions the equation has.

Show answer

a = 2, b = 3, c = 5.
D = 3² − 4·2·5 = 9 − 40 = −31.
D < 0 → no real solutions (solutions are complex).

STAAR-Style Practice

1. What is the discriminant of x² + 4x + 4 = 0?

  1. −12
  2. 0
  3. 4
  4. 16

2. The quadratic equation has discriminant D > 0. What does this tell you?

  1. There is one real solution.
  2. There are two real solutions.
  3. There are no real solutions.
  4. The equation is linear, not quadratic.

3. For which equation is the discriminant negative?

  1. x² − 4x + 4 = 0
  2. x² − 5x + 6 = 0
  3. x² + 2x + 5 = 0
  4. x² − 3x = 0
Show STAAR-style answers and explanations

1. a = 1, b = 4, c = 4 → D = 4² − 4·1·4 = 16 − 16 = 0 → B.

2. D > 0 → two distinct real solutions → B.

3. For x² + 2x + 5: D = 2² − 4·1·5 = 4 − 20 = −16 < 0 → C.

The discriminant is a quick “solution checker” — it tells you what kind of roots to expect before you do all the work.
