Key Idea (Quadratic Formula):
Any quadratic ax² + bx + c = 0 can be solved with
x = [−b ± √(b² − 4ac)] / (2a).

Example (General Algebra)

Solve 2x² − 3x − 5 = 0 using the quadratic formula.

Step 1 – Identify a, b, c:

  • a = 2, b = −3, c = −5

Step 2 – Plug into the formula:

  • x = [−(−3) ± √((−3)² − 4·2·(−5))] / (2·2)
  • x = [3 ± √(9 + 40)] / 4
  • x = [3 ± √49] / 4
  • x = [3 ± 7] / 4

Step 3 – Find both solutions:

  • x = (3 + 7) / 4 = 10 / 4 = 5/2
  • x = (3 − 7) / 4 = −4 / 4 = −1

Solutions: x = 5/2 and x = −1.

Your Turn (General Algebra)

Solve x² + 4x − 5 = 0 using the quadratic formula.

Show answer

a = 1, b = 4, c = −5.
x = [−4 ± √(16 − 4·1·(−5))] / 2 = [−4 ± √(16 + 20)] / 2 = [−4 ± √36] / 2 = [−4 ± 6]/2.
x = (−4 + 6)/2 = 2/2 = 1; x = (−4 − 6)/2 = −10/2 = −5.
Solutions: x = 1, x = −5.

STAAR-Style Practice

1. For the equation 3x² + 2x − 1 = 0, what are the values of a, b, and c?

  1. a = 3, b = 2, c = −1
  2. a = 3, b = −2, c = 1
  3. a = 2, b = 3, c = −1
  4. a = 1, b = 2, c = 3

2. Which quadratic equation could be solved using the quadratic formula?

  1. x + 5 = 7
  2. 2x − 3 = 0
  3. 4x² − x − 2 = 0
  4. |x − 3| = 5

3. When using the quadratic formula, what part tells you how many solutions you will get?

  1. a
  2. b
  3. c
  4. b² − 4ac
Show STAAR-style answers and explanations

1. Equation is already in ax² + bx + c form → a = 3, b = 2, c = −1 → A.

2. Only a true quadratic (x² term) uses the formula → 4x² − x − 2 = 0 → C.

3. The discriminant b² − 4ac controls the number and type of solutions → D.

The quadratic formula always works—even when factoring is messy—just be careful with negatives and the square root.
