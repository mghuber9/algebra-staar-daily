Goal: Practice skills with Vertex From Graph.

Key Idea (Vertex From Graph):
The vertex is the highest or lowest point on a parabola. If it opens up, the vertex is a minimum; if it opens down, the vertex is a maximum.

Example (General Algebra)

Suppose the graph of a quadratic passes through (0, 4), (1, 1), and has its lowest point at (2, 0), then rises again through (3, 1).

Observation:

  • The point (2, 0) is the bottom of the U-shape.
  • The parabola opens upward.

Vertex: (2, 0), and it represents a minimum value of 0 for y.

Your Turn (General Algebra)

A parabola opens downward and has its highest point at (−1, 5). What is the vertex, and is it a maximum or minimum?

Show answer

Vertex: (−1, 5).
Since the parabola opens down, this point is a maximum.

STAAR-Style Practice

1. The graph of y = f(x) has a vertex at (3, −4) and opens upward. Which statement is true?

  1. The maximum value of the function is −4.
  2. The minimum value of the function is −4.
  3. The maximum value of the function is 3.
  4. The minimum value of the function is 3.

2. The parabola y = g(x) has a vertex at (0, 6) and opens downward. Which ordered pair represents the vertex?

  1. (0, −6)
  2. (6, 0)
  3. (0, 6)
  4. (−6, 0)

3. A quadratic function models the height (in meters) of a ball over time. The vertex of its graph is at (1.2, 8). What does the y-value of the vertex represent?

  1. The starting height of the ball
  2. The highest height of the ball
  3. The time when the ball hits the ground
  4. The speed of the ball when thrown
Show STAAR-style answers and explanations

1. Opens up → vertex is a minimum → y = −4 is the minimum value → B.

2. The vertex is given directly as (0, 6) → C.

3. In a height vs. time graph, the vertex is the highest point → maximum height: 8 meters → B.

When reading a parabola, focus on the vertex’s coordinates: x is “when” (or input), y is the “best/worst” value (maximum or minimum).
