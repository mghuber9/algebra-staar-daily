Day 74 – Vertex Using –b/2a

Goal: Practice skills with Vertex Using –b/2a.

← Back to all lessons
Key Idea (Vertex Using −b/2a):
For a quadratic in standard form y = ax² + bx + c, the x-coordinate of the vertex is x = −b / (2a). Plug that x back into the function to find the y-coordinate.

Example (General Algebra)

Find the vertex of y = 2x² − 8x + 1.

Step 1 – Find x-coordinate of vertex:

  • a = 2, b = −8
  • x = −b / (2a) = −(−8) / (2·2) = 8 / 4 = 2

Step 2 – Plug x = 2 into the equation:

  • y = 2(2)² − 8(2) + 1 = 2·4 − 16 + 1 = 8 − 16 + 1 = −7

Vertex: (2, −7).

Your Turn (General Algebra)

Find the vertex of y = −x² + 4x + 1.

Show answer

a = −1, b = 4 → x = −b / (2a) = −4 / (−2) = 2.
y = −(2)² + 4(2) + 1 = −4 + 8 + 1 = 5.
Vertex: (2, 5).

STAAR-Style Practice

1. What is the x-coordinate of the vertex of y = x² − 6x + 2?

  1. −3
  2. 3
  3. −6
  4. 6

2. The function y = −2x² + 8x − 1 models the height of a ball. What is the x-coordinate of the vertex?

  1. −2
  2. 0
  3. 2
  4. 4

3. For y = x² + 2x + 5, which ordered pair is the vertex?

  1. (−1, 4)
  2. (−1, 6)
  3. (1, 4)
  4. (1, 6)
Show STAAR-style answers and explanations

1. a = 1, b = −6 → x = −(−6)/(2·1) = 6/2 = 3 → B.

2. a = −2, b = 8 → x = −8/(2·−2) = −8/−4 = 2 → C.

3. a = 1, b = 2 → x = −2/2 = −1. Then y = (−1)² + 2(−1) + 5 = 1 − 2 + 5 = 4 → vertex (−1, 4) → A.

−b/2a is just a shortcut for the axis of symmetry; the vertex always lies on this line.
