Day 59 – Factored Form

Goal: Practice skills with Factored Form.

← Back to all lessons
Key Idea (Factored Form):
Factored form of a quadratic is y = a(x − r₁)(x − r₂). The x-intercepts are r₁ and r₂, and the axis of symmetry is x = (r₁ + r₂) / 2.

Example (General Algebra)

Consider y = (x − 1)(x − 5).

Step 1 – Find x-intercepts:

  • Set each factor = 0: x − 1 = 0 → x = 1; x − 5 = 0 → x = 5.
  • x-intercepts: (1, 0) and (5, 0).

Step 2 – Axis of symmetry:

  • x = (1 + 5) / 2 = 6 / 2 = 3.

Step 3 – Vertex (idea only): The vertex lies on the axis of symmetry; you could plug in x = 3 to find its y-value.

Your Turn (General Algebra)

Find the x-intercepts of y = 2(x + 4)(x − 1).

Show answer

x + 4 = 0 → x = −4; x − 1 = 0 → x = 1.
x-intercepts: (−4, 0) and (1, 0).

STAAR-Style Practice

1. What are the x-intercepts of y = (x − 2)(x + 6)?

  1. (−2, 0) and (6, 0)
  2. (2, 0) and (−6, 0)
  3. (2, 0) only
  4. (−6, 0) only

2. For y = −(x − 4)(x − 8), what is the axis of symmetry?

  1. x = 4
  2. x = 8
  3. x = 6
  4. x = 12

3. A quadratic in factored form is y = (x + 1)(x + 9). Which statement is true?

  1. The x-intercepts are (1, 0) and (9, 0).
  2. The x-intercepts are (−1, 0) and (−9, 0).
  3. The axis of symmetry is x = 10.
  4. The axis of symmetry is x = −5.
Show STAAR-style answers and explanations

1. x − 2 = 0 → x = 2; x + 6 = 0 → x = −6 → x-intercepts (2, 0) and (−6, 0) → B.

2. r₁ = 4, r₂ = 8 → axis x = (4 + 8) / 2 = 6 → C.

3. x + 1 = 0 → x = −1; x + 9 = 0 → x = −9 (so x-intercepts are negative). Axis x = (−1 + (−9)) / 2 = −10/2 = −5 → statement “axis is x = −5” is true → D.

Factored form is the fastest way to read off x-intercepts and the axis of symmetry—great for graph sketches and intercept questions.
