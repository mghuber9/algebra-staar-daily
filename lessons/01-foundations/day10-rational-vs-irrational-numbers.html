Day 10 – Rational vs Irrational Numbers

Goal: Practice skills with Rational vs Irrational Numbers.

← Back to all lessons
Key Idea:
• A rational number can be written as a fraction of integers (a/b, b ≠ 0). Its decimal form terminates or repeats.
• An irrational number cannot be written as a fraction of integers. Its decimal form is non-terminating and non-repeating.

Example (General Algebra)

Decide whether each number is rational or irrational:

  • 0.75
  • √2

0.75 = 75/100, which is a fraction of integers → rational.

√2 has a decimal that goes on forever without repeating → irrational.

Your Turn (General Algebra)

Classify each as rational or irrational:

  • −5
  • π
Show answer

−5 = −5/1 → rational.
π is non-terminating, non-repeating → irrational.

STAAR-Style Practice

1. Which number is irrational?

  1. 3.5
  2. 7/9
  3. √16
  4. √5

2. Which statement about the number 0.121212... (where "12" repeats) is true?

  1. It is irrational because it does not end.
  2. It is rational because it can be written as a fraction.
  3. It is irrational because it is a decimal.
  4. It is rational because it is a whole number.

3. A student claims that √49 is irrational because it involves a square root. Which explains why the student is incorrect?

  1. √49 is irrational because it is a decimal.
  2. √49 is rational because 49 is a whole number.
  3. √49 is rational because it equals 7, which is a whole number.
  4. √49 is irrational because 7 is not a fraction.
Show STAAR-style answers and explanations

1. √5 is not a perfect square, and its decimal is non-terminating and non-repeating → D is irrational.
The others are all rational.

2. 0.121212... is a repeating decimal, so it can be written as a fraction → B (rational).

3. √49 = 7, and 7 is a whole number, which is rational → C.
Having a square root does not automatically make a number irrational.

Connect to STAAR: Students should quickly recognize perfect squares as rational and π, √2, √5, etc. as irrational.
