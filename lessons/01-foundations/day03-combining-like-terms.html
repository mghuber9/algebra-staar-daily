Day 03 – Combining Like Terms

Goal: Practice skills with Combining Like Terms.

← Back to all lessons
Key Idea:
Like terms have the same variable part (same variable and same exponent). You can combine them by adding or subtracting their coefficients.

Example (General Algebra)

Simplify 5x + 3x − 4.

Step 1 – Combine like terms with x: 5x + 3x = 8x

Step 2 – Bring down the constant: 8x − 4

The simplified expression is 8x − 4.

Your Turn (General Algebra)

Simplify 7y − 2y + 9.

Show answer

7y − 2y = 5y → 5y + 9. The simplified expression is 5y + 9.

STAAR-Style Practice

1. Which expression is equivalent to 4x + 2 − x?

  1. 3x + 2
  2. 5x + 2
  3. 4x − x − 2
  4. 3x − 2

2. Simplify the expression 6a + 4 − 3a + 1.

  1. 9a + 5
  2. 3a + 5
  3. 3a + 3
  4. −3a + 5

3. Which expression is equivalent to 2x + 5x − 3 + 4?

  1. 7x − 7
  2. 7x + 1
  3. 9x + 1
  4. 9x − 1
Show STAAR-style answers and explanations

1. 4x + 2 − x → (4x − x) + 2 = 3x + 2 → A.

2. 6a + 4 − 3a + 1 → (6a − 3a) + (4 + 1) = 3a + 5 → B.

3. 2x + 5x − 3 + 4 → (2x + 5x) + (−3 + 4) = 7x + 1 → B.
Watch for sign mistakes when combining constants.

Hint for students: highlight or circle like terms before you combine them.
