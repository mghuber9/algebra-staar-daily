Day 51 – Infinite / No Solution

Goal: Practice skills with Infinite / No Solution.

← Back to all lessons
Key Idea (Infinite / No Solution):
When solving an equation, you can get:
  • One solution – a specific value of x.
  • Infinite solutions – both sides are always equal (identity).
  • No solution – you get a false statement like 3 = 5.

Example (General Algebra)

Decide if this equation has one solution, infinite solutions, or no solution:

2(x + 3) = 2x + 6

Step 1 – Distribute:

  • 2(x + 3) = 2x + 6
  • 2x + 6 = 2x + 6

Step 2 – Subtract 2x from both sides:

  • 2x + 6 − 2x = 2x + 6 − 2x
  • 6 = 6

Conclusion: 6 = 6 is always true, no matter what x is. The equation has infinitely many solutions.

Your Turn (General Algebra)

Does the equation 3(x − 1) = 3x + 2 have one solution, no solution, or infinitely many solutions?

Show answer

Distribute: 3x − 3 = 3x + 2.
Subtract 3x: −3 = 2.
This is false, so there is no solution.

STAAR-Style Practice

1. Solve: 4x + 8 = 4(x + 2)

  1. x = 0
  2. x = 2
  3. Infinite solutions
  4. No solution

2. Which equation has no solution?

  1. 5x − 2 = 5x − 2
  2. 2(x + 4) = 2x + 8
  3. 3x + 5 = 3x − 1
  4. x + 7 = 2x + 1

3. Which equation has infinitely many solutions?

  1. 2x + 1 = 5
  2. 6x − 3 = 6x − 3
  3. 4x + 2 = 4x − 6
  4. 3(x + 1) = 3x + 4
Show STAAR-style answers and explanations

1. 4x + 8 = 4x + 8 → 8 = 8, always true → infinite solutions → C.

2. 3x + 5 = 3x − 1 → subtract 3x: 5 = −1 (false) → no solution → C.

3. 6x − 3 = 6x − 3 is true for all x → infinite solutions → B.

After simplifying, if you see “number = number” (like 4 = 4), that’s infinite solutions. If you see “number = different number” (like 4 = 6), that’s no solution.
