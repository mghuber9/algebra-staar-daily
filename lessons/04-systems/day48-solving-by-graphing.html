Day 48 – Solving by Graphing

Goal: Practice skills with Solving by Graphing.

← Back to all lessons
Key Idea (Solving by Graphing):
To solve a system by graphing, graph both equations on the same coordinate plane and find the intersection point. That point is the solution.

Example (General Algebra)

Solve the system by graphing:

  • y = x + 1
  • y = −x + 5

Step 1 – Graph y = x + 1:

  • Slope 1, y-intercept 1 → start at (0, 1), go up 1, right 1.

Step 2 – Graph y = −x + 5:

  • Slope −1, y-intercept 5 → start at (0, 5), go down 1, right 1.

Step 3 – Locate the intersection: The lines cross at (2, 3).

Check:

  • In y = x + 1 → 3 = 2 + 1 ✔
  • In y = −x + 5 → 3 = −2 + 5 ✔

The solution is (2, 3).

Your Turn (General Algebra)

Suppose the system

  • y = 2x − 1
  • y = −x + 5

is graphed and the lines intersect at (2, 3). What is the solution?

Show answer

The intersection point is the solution.
Answer: (2, 3).

STAAR-Style Practice

1. A graph shows the intersection of two lines at (−1, 4). Which ordered pair is the solution to the system?

  1. (4, −1)
  2. (−1, 4)
  3. (1, −4)
  4. (−4, 1)

2. When solving a system by graphing, the lines intersect at (0, −2). Which statement is true?

  1. Both equations are true when x = 0 and y = −2.
  2. Only one equation is true when x = 0 and y = −2.
  3. Neither equation is true when x = 0 and y = −2.
  4. The system has no solution.

3. What does it mean if the lines in a graphed system are parallel?

  1. The system has exactly one solution.
  2. The system has no solution.
  3. The system has exactly two solutions.
  4. The system has infinitely many solutions.
Show STAAR-style answers and explanations

1. The solution is the intersection point → (−1, 4) → B.

2. At the intersection, both equations are true for that ordered pair → A.

3. Parallel lines never intersect → no solution → B.

Graphing is a great way to see the solution visually and to tell if there is one solution, no solution, or infinitely many solutions.
