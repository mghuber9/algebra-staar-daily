Key Idea (System Solutions From Table):
When given tables for a system, look for an ordered pair (x, y) that appears in both tables. That common pair is the solution to the system.

Example (General Algebra)

Two functions are represented by tables:

Table for y = f(x):

  • (0, 3), (1, 5), (2, 7), (3, 9)

Table for y = g(x):

  • (0, 1), (1, 3), (2, 5), (3, 7)

Step 1 – Compare ordered pairs: Look for a pair that appears in both lists.

  • Common point: (none yet) in this example.

For a better illustration, suppose the second table is:

  • (0, 1), (1, 3), (2, 7), (3, 11)

Now the point (2, 7) appears in both tables, so (2, 7) is the solution.

Your Turn (General Algebra)

Find the solution to the system using these tables:

Table A: (−1, 0), (0, 2), (1, 4), (2, 6)

Table B: (−1, 4), (0, 3), (1, 4), (2, 5)

Show answer

Compare pairs: only (1, 4) appears in both tables.
Solution: (1, 4).

STAAR-Style Practice

1. Which ordered pair is a solution to both equations if their tables contain the following?

Equation 1: (0, 1), (1, 4), (2, 7), (3, 10)

Equation 2: (0, 5), (1, 4), (2, 3), (3, 2)

  1. (0, 1)
  2. (1, 4)
  3. (2, 7)
  4. (3, 10)

2. A system of equations is represented by two tables. The only ordered pair that appears in both tables is (4, −2). What is the solution to the system?

  1. x = 4 only
  2. y = −2 only
  3. (4, −2)
  4. There is no solution

3. If two tables for a system of equations have no ordered pairs in common, what can you conclude?

  1. The system has one solution.
  2. The system has no solution.
  3. The system has exactly two solutions.
  4. The system has infinitely many solutions.
Show STAAR-style answers and explanations

1. Compare lists: (1, 4) is in both → B.

2. The common ordered pair is the solution to the system → (4, −2) → C.

3. No common ordered pair means no (x, y) works in both equations → no solution → B.

Tables are just lists of (x, y) pairs—finding a solution to a system is a “match-the-pair” game between the two lists.
