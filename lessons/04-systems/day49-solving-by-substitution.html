Day 49 – Solving by Substitution

Goal: Practice skills with Solving by Substitution.

← Back to all lessons
Key Idea (Substitution):
In substitution, solve one equation for a variable, then plug that expression into the other equation. This turns the system into a single equation with one variable.

Example (General Algebra)

Solve the system:

  • y = 2x + 1
  • x + y = 10

Step 1 – Substitute y from the first equation into the second:

  • x + (2x + 1) = 10

Step 2 – Combine like terms:

  • 3x + 1 = 10

Step 3 – Solve for x:

  • 3x = 9 → x = 3

Step 4 – Plug x back in to find y:

  • y = 2(3) + 1 = 7

The solution is (3, 7).

Your Turn (General Algebra)

Solve the system using substitution:

  • y = x − 4
  • 2x + y = 8
Show answer

Substitute y = x − 4 into 2x + y = 8:
2x + (x − 4) = 8 → 3x − 4 = 8 → 3x = 12 → x = 4.
Then y = x − 4 = 4 − 4 = 0.
Solution: (4, 0).

STAAR-Style Practice

1. Solve the system:

  • y = 3x
  • x + y = 8
  1. (1, 7)
  2. (2, 6)
  3. (3, 5)
  4. (4, 4)

2. The system is given by y = 5 − x and 2x + y = 7. What is the solution?

  1. (1, 4)
  2. (2, 3)
  3. (3, 2)
  4. (4, 1)

3. Which situation is easiest to solve with substitution?

  1. Both equations are in standard form (Ax + By = C).
  2. One equation is already solved for a variable (y = …).
  3. Both equations have the same x-coefficient.
  4. Both equations have the same y-coefficient.
Show STAAR-style answers and explanations

1. Substitute y = 3x into x + y = 8 → x + 3x = 8 → 4x = 8 → x = 2, y = 6 → (2, 6) → B.

2. Substitute y = 5 − x into 2x + y = 7:
2x + (5 − x) = 7 → x + 5 = 7 → x = 2, y = 3 → (2, 3) → B.

3. Substitution works best when one equation already has y = … or x = … → B.

Substitution is your go-to when you see an equation like y = … or x = … because you can plug it directly into the other equation.
