Key Idea (System Solutions From Graph):
A solution to a system of linear equations is the point where the lines intersect. On a graph, look for the point that lies on both lines.

Example (General Algebra)

Consider the system:

  • y = 2x + 1
  • y = −x + 7

Graph idea: When you graph both lines, they intersect at the point (2, 5).

Check the intersection point:

  • First equation: y = 2(2) + 1 = 5 ✔
  • Second equation: y = −2 + 7 = 5 ✔

So the solution to the system is (2, 5).

Your Turn (General Algebra)

A graphed system shows two lines crossing at (−1, 4). What is the solution to the system?

Show answer

The solution is simply the intersection point.
Answer: (−1, 4).

STAAR-Style Practice

1. On a coordinate plane, two lines intersect at (3, −2). Which statement is true?

  1. (3, −2) is a solution to exactly one of the equations.
  2. (3, −2) is a solution to neither equation.
  3. (3, −2) is a solution to both equations.
  4. (3, −2) is not a possible solution to any system.

2. A system is graphed and the lines do not intersect; they are parallel. What can you conclude?

  1. The system has exactly one solution.
  2. The system has no solution.
  3. The system has infinitely many solutions.
  4. The system has two solutions.

3. A system is graphed and the two lines are the same (on top of each other). What can you conclude?

  1. No solution
  2. Exactly one solution
  3. Exactly two solutions
  4. Infinitely many solutions
Show STAAR-style answers and explanations

1. A solution to a system lies on both lines, so (3, −2) is on both → C.

2. Parallel lines never meet → no intersection → no solution → B.

3. If the lines are the same, every point on the line is a solution → infinitely many solutions → D.

For systems, always think “where do the lines meet?”—that point is the shared (x, y) that works in both equations.
