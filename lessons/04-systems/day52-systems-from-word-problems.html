Day 52 – Systems From Word Problems

Goal: Practice skills with Systems From Word Problems.

Key Idea (Systems From Word Problems):
For word problems with two unknowns, define two variables, write two equations, and solve the system (by graphing, substitution, or elimination).

Example (General Algebra)

A movie theater sold 80 tickets for a show. Adult tickets cost $10, and student tickets cost $6. The total money collected was $640. How many of each type were sold?

Step 1 – Define variables:

  • Let a = number of adult tickets.
  • Let s = number of student tickets.

Step 2 – Write equations:

  • Total tickets: a + s = 80
  • Total money: 10a + 6s = 640

Step 3 – Solve the system (use substitution/elimination):

  • From a + s = 80 → a = 80 − s.
  • Substitute into money equation: 10(80 − s) + 6s = 640
  • 800 − 10s + 6s = 640 → 800 − 4s = 640 → −4s = −160 → s = 40
  • Then a = 80 − 40 = 40

Answer: 40 adult tickets and 40 student tickets.

Your Turn (General Algebra)

A school sold 120 shirts. Short-sleeve shirts cost $12 and long-sleeve shirts cost $18. The total money collected was $1740. Let s = short-sleeve shirts and l = long-sleeve shirts. Write a system of equations that models the situation.

Show answer

Total shirts: s + l = 120.
Total money: 12s + 18l = 1740.

STAAR-Style Practice

1. A snack bar sells popcorn for $3 and drinks for $2. In one hour, they sold 25 items for a total of $64. Let p = popcorn and d = drinks. Which system represents this situation?

  1. p + d = 25 and 3p + 2d = 64
  2. p + d = 64 and 3p + 2d = 25
  3. 3p + d = 25 and 2p + d = 64
  4. p + d = 25 and 2p + 3d = 64

2. A student buys 3 pencils and 2 notebooks for $7.50. Another student buys 2 pencils and 4 notebooks for $11.00. Let x = price of a pencil and y = price of a notebook. Which system models this?

  1. 3x + 2y = 7.50 and 2x + 4y = 11.00
  2. 3x + 2y = 11.00 and 2x + 4y = 7.50
  3. 2x + 3y = 7.50 and 4x + 2y = 11.00
  4. x + y = 7.50 and x − y = 11.00

3. In a word problem that asks you to “find the number of adults and children” at an event, what is usually the best first step?

  1. Graph a line with slope 2.
  2. Guess and check until you find numbers that work.
  3. Define variables and write equations.
  4. Multiply both sides of an equation by 0.
Show STAAR-style answers and explanations

1. Total items → p + d = 25; total cost → 3p + 2d = 64 → A.

2. 3 pencils, 2 notebooks → 3x + 2y = 7.50; 2 pencils, 4 notebooks → 2x + 4y = 11.00 → A.

3. Systems word problems start with defining variables and writing equations → C.

Almost every systems word problem is “number of items” + “total cost/total value” → one equation for each.
